Creación del Sitio Web "Casa de Materiales"

Efraín, James, Alfonso, Ángel, Enrique y Anhelo hicieron equipo para crear un sitio web para "Casa de Materiales", dedicada a la venta de materiales de construcción.

Este proyecto surgió con la intención de un proyecto integrador de nuestra carrera, "Programación".

Desarrollo del Sitio Web

Para crear la pagina "Casa de Materiales" se utilizó HTML y CSS para desarrollar la estructura y el diseño del sitio web.Nos enfocamos en crear una interfaz intuitiva y fácil de navegar, destacando los materiales disponibles.

Además, se integraron imágenes de alta calidad y contenido informativo sobre cada tipo de material.

Lanzamiento y Resultados

El equipo actualizara y mejorara el sitio web para adaptarse a las necesidades cambiantes del mercado.
